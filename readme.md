<h2>Procedure to push on 2 remotes</h2>

Create key origin
```
git remote add origin SSH-REMOTE-URL-1
```

Then add url's:
```
git remote set-url --add --push origin REMOTE-URL-1
```
```
git remote set-url --add --push origin REMOTE-URL-2
```

Finally, it remains only to push
```
git push
```
